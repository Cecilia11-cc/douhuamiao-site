我的手作项目

"让人'啊'一声的科学问题" 闯关小游戏 游戏教育

一个面向小学中高年级的科学思维教育互动网站，用"看到奇怪现象 → 追问为什么"的方式，培养好奇心与科学探究精神。

双模式体验

  • 游客端 — 无需密码，直接闯关答题，体验四种问题类型的思维挑战
  • 教师演示端 — 输入密码进入，逐步揭示问题背景、解题思路和完整答案，适合课堂教学演示

四道关卡

  • 情境问题 — 生活中的奇怪现象：费曼的水杯取硬币、一只手喝倒扣水杯的水……从"这怎么可能？！"出发，感受观察现象 → 追问本质的探究过程
  • 语言问题 — 文字的奥秘：标点符号谜题、语序差异、一词多义，体会语言表达的精确性与丰富性，原来科学也需要"会说话"
  • 策略问题 — 量水与分药：5L 和 3L 水壶量出 4L 水的互动操作谜题，在多种方案中寻找最优解。教师端还有经典分药难题的三步拆解，让孩子体验"思考如何思考"
  • 逻辑问题 — 骑士与无赖：两条路、两个人、一个关键问题。通过角色扮演和表格分析，破解逻辑谜题，感受演绎推理的力量

设计特色

  • 仙桃田园风格 Canvas 动态背景：远山、桃树、花瓣飘落
  • 童趣化卡片界面，彩虹渐变装饰，适合小学生使用
  • 响应式布局，适配电脑、平板和手机

内容来源与致谢

本游戏的问题设计，部分改编与参考自：

  • 本人"三下乡"支教磨课期间制作的教学 PPT
  • 《浴缸里的惊叹：256 道让你恍然大悟的趣题》（顾森 著）

在此对顾森老师表达诚挚的感谢。书中的精彩趣题为本游戏提供了核心的思维训练框架与灵感。

拼豆图纸生成器 前端创意

大学美育课要交手工作品，我没什么好手艺，打算拼豆。但碰上了比赛，时间紧，想请人帮忙，结果一问——拼豆设计稿要 50 块钱。算了。
刚好我的 Matlab 又坏了，干脆把 slandarer 大佬的 Matlab 拼豆图纸生成代码改成了 Python 版。

现在，把任意图片一键转换为拼豆图纸，内置 Mard 221 色标准色卡，自动限制颜色数量，输出带色号和网格的像素对照图，非常适合手工拼豆制作参考。

功能亮点

  • 支持任意尺寸的 JPG / PNG 图片，自动居中裁剪为正方形
  • 内置 Mard 拼豆官方 221 色卡，颜色还原度高
  • 可自定义像素尺寸（默认 50×50）和最大使用颜色数
  • 输出高清图纸，包含：带网格和坐标的像素图、每个像素块上标注的色号、右侧图例（色号、颜色块和所需豆子数量）
  • 纯 Python 实现，无需额外软件

Python 3 · NumPy · Matplotlib · Pillow

灵感与致谢

本项目的核心图像转换算法思路，受到了以下优秀作品的启发：

  • 原作者：MATLAB-pairplot（GitHub）
  • 微信公众号：slandarer 随笔
  • 文章：《MATLAB | 如何使用 MATLAB 生成拼豆图纸》（2025-11-11）

我重新用 Python 独立实现了拼豆图纸生成流程，并在 AI 辅助下进行了大量重构和功能扩展。颜色数据来源于公开的拼豆色库，本人独立整理并收录了 221 种颜色。

PDF → LaTeX 转换工具 via DeepSeek Python工具

打比赛的时候我负责写论文，需要根据比赛模板来设计论文框架。可我还是 LaTeX 新手，对着模板发愁——忽然想到，能不能用 API 扫描模板，让模型帮我理解结构？

结果真的成了。虽然字体、表格、图片的处理还很粗糙，但这个脚本已经能帮我把一篇 PDF 论文自动转成结构清晰的 LaTeX 草稿，省下了大量从零搭建框架的时间。

一个 Python 脚本，调用 DeepSeek 大模型，将中文数学建模论文 PDF 自动转换为结构清晰的 LaTeX 文档。

能做什么

  • 提取 PDF 中的全部文字（支持多页）
  • 自动生成符合中文排版习惯的 LaTeX 文件（ctexart）
  • 将原文中的公式转为 LaTeX 数学环境
  • 表格转为三线表（booktabs）
  • 保留图片、参考文献的占位符
  • 本地运行，你的 PDF 不会上传到任何地方（仅将提取的文字发送给 DeepSeek API）
已知限制（使用前必读）
本项目是一个辅助工具，并非完美的一键转换方案。表格会被提取为纯文字需手动核对；图片转为占位符需手动导出插入；字体使用 ctexart 默认字体与原文档有差异。建议将输出视为"粗排草稿"，在此基础上手动精修，远比从零写起高效得多。
