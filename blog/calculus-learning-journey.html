一、微积分基本定理：从定理到反例

1.1 定理本身

微积分基本定理分为两部分：

第一部分（原函数存在定理）

若 f 在 [a,b] 上连续，则积分上限函数

Φ(x) = ∫ax f(t) dt

可导，且 Φ'(x) = f(x)。即：连续函数必有原函数。

第二部分（牛顿-莱布尼茨公式）

若 F' = f，则

∫ab f(x) dx = F(b) − F(a)

1.2 当连续性被破坏：两个反例

反例一：可去间断点——导数存在但不等于函数值

设 f(t) = 1（t≠0），f(0) = 0。积分上限函数 Φ(x) = ∫0x f(t)dt = x，故 Φ'(x) = 1 处处成立。但在 x=0 处：

Φ'(0) = 1 ≠ 0 = f(0)

结论：改变一个点不改变积分值，但破坏了导数与被积函数的相等关系。

反例二：跳跃间断点——导数根本不存在

设 f(t) = −1（t<0），f(0)=0，f(t) = 1（t>0）。则 Φ(x) = |x|，在 x=0 处不可导（左右导数分别为 −1 和 1）。

共同教训："f 连续"是 Φ'(x) = f(x) 的关键条件。一旦连续性被破坏，结论就可能失效。

1.3 反过来成立吗？

若 Φ'(x) = f(x) 处处成立，f 是否必连续？不一定。

经典反例：f(x) = sin(1/x)（x≠0），f(0)=0。f 在 0 处有震荡间断点，但可证 Φ'(0) = 0 = f(0)。

f 连续是 Φ' = f 的充分条件，但不是必要条件。导函数即使不连续，也必定满足达布定理（介值性），而连续是一种更强的性质。

二、黎曼可积：条件全梳理

2.1 充要条件（最本质的刻画）

勒贝格准则 (Lebesgue's Criterion)

f 在 [a,b] 上黎曼可积 ⇔ f 在 [a,b] 上有界，且其间断点集的勒贝格测度为 0（即"几乎处处连续"）。

2.2 常用充分条件（由窄到宽）

  1. 连续函数：闭区间上的连续函数必可积。
  2. 只有有限个间断点的有界函数：无论可去还是跳跃间断点。
  3. 间断点可数且有界：如有理点处间断、无理点处连续的函数。
  4. 单调函数：即使有无穷多个跳跃间断点，也必有界且间断点至多可数。
  5. 几乎处处连续的有界函数：间断点集为零测度集（如康托集）。

2.3 可积函数类的包含关系

连续函数 ⊂ 有限个间断点的有界函数 ⊂ 单调函数 ⊂ 间断点可数且有界 ⊂ 几乎处处连续的有界函数（充要条件）

2.4 积分上限函数的行为总结

  • 所有可积函数的 Φ(x) = ∫ax f(t)dt 都存在且连续。
  • 在 f 的连续点处：Φ'(x) = f(x)。
  • 在 f 的跳跃间断点处：Φ 不可导（尖点）。
  • 在 f 的可去间断点处：Φ 可导，但导数等于极限值而非 f 的值本身。

三、勒贝格测度：当数学遇见直觉

3.1 两种积分的根本区别

核心直觉

黎曼积分"竖着切"定义域（x 轴），勒贝格积分"横着切"值域（y 轴）。

黎曼积分：把 [a,b] 切成小区间 Δxi，算小矩形面积 f(xi*)Δxi。

勒贝格积分：把函数值范围切成小段 [yi, yi+1]，回头找使 f(x) 落在这个小段里的所有 x，看这些 x 构成的点集"有多大"（测度），再乘高度。

3.2 测度到底是什么？

测度就是"长度/面积/体积"概念的终极数学化推广。

  • [0,1] 的勒贝格测度 = 1（和长度一致）
  • 单点集的测度 = 0
  • [0,1] 里所有有理数构成的集合，测度 = 0（可数个 0 加起来还是 0）
  • 康托集，虽然不可数，但测度也 = 0

3.3 概率与测度：本质上是一个东西

在几何概型里：概率 = 某个区域的测度 / 总区域的测度。

一个点没有面积，所以丢芝麻丢到某个特定坐标点的概率是 0。概率为 0 的事件，未必是不可能事件。这正是测度论给概率论带来的概念升级。

3.4 延伸思考：测度与人生的隐喻

"孤立的点对一维世界没有贡献。"——这几乎是勒贝格测度的一句完美格言。

黎曼可积的条件告诉我们：只要"不连续"的部分总长度为零，就无碍大局。对应到学习上：

  • 偶尔的懈怠（有限个间断点）→ 不影响积分值。
  • 整段整段时间的放弃（正测度的间断）→ 对积分有可察觉的负面影响。
  • 真正起决定作用的，不是你每分每秒都在"动作"，而是你"不动作"的那些点究竟有多少。
数学安慰

人生的"积分"取决于你"持续努力"的区间长度。孤立的懈怠点（无论多少个，只要总长度为零）都改变不了最终的积分值。偶尔放空一下、失误一下，在数学上是完全可以被原谅的。

四、连续统：点如何质变成线

4.1 两种不同的"无限"

类型 基数 测度 举例
可数无限 ℵ₀ 0 自然数、有理数
连续统 2ℵ₀ 正数（如 1） 实数、任意区间上的全体实数

4.2 关键命题

不可数个零测度集合的并集，测度可以大于零。勒贝格测度只保证"可数可加性"，不保证"不可数可加性"。

[0,1] 区间由不可数个测度为 0 的点组成，但总测度是 1。这不是靠点的"累积叠加"，而是靠连续统的"厚度"——一种本体论意义上的跃迁。

4.3 "连续"与"无限"不可分割

在标准数学里，只要一个集合是连续的（连通的），它内部就必须包含不可数无限个点。有限个点必然是离散的，可数无限个点依然"处处漏风"。

一旦你真正触及了连续的质感，你就已经身在无限的洪流之中了。

4.4 学习的隐喻升级

  • 有限次努力：测度为 0。
  • 可数无限次努力（每天固定整点学一小时）：次数无限，但总"长度"依然为 0——可数个点凑不出长度。
  • 连续统式的沉浸：无法拆解成孤立瞬间的"心流"，整个时间段被学习填满——这段区间就是连续统，拥有了正的"厚度"与贡献。

五、多项式因式分解：长除法、综合除法与技巧库

5.1 长除法的本质

和整数竖式除法完全一样，只是把"位"换成了"次数"，把"10 的幂"换成了"x 的幂"。

口诀：消最高次 → 乘回去 → 相减 → 降一位，循环。

5.2 缺项处理（最易错点）

如 x⁴ − 16 除以 x − 2，必须补零占位：x⁴ + 0x³ + 0x² + 0x − 16，再进行长除。不补零会导致竖式错位，后面全错。

5.3 综合除法——长除法的压缩版

专门用于除式为 x − c 的情况。只写系数，做"落、乘、加"循环。

例：2x³ − 3x² − 3x + 2 除以 x + 1（即 c = −1） -1 │ 2 -3 -3 2 │ -2 5 -2 └───────────────── 2 -5 2 │ 0 结果：商式 2x² − 5x + 2，余数 0（整除）

原理：本质是待定系数法的速算形式。每一步 bi−1 = ai + c·bi 来自系数的恒等关系。

5.4 因式分解武器库

  1. 分组分解：奇偶项分开、前两组+后两组、三一分组。
  2. 添拆项法：如 x⁴ + 4 = x⁴ + 4x² + 4 − 4x² = (x²+2)² − (2x)²。
  3. 待定系数法：已知形式时设出因式，对比系数解方程。
  4. 换元法：看到重复结构就设 t = 某表达式。
  5. 平方差/立方和差公式推广：aⁿ − bⁿ = (a−b)(aⁿ⁻¹ + aⁿ⁻²b + … + bⁿ⁻¹)。

5.5 两个核心定理

余数定理 (Remainder Theorem)

多项式 P(x) 除以 (x − c)，所得余数等于 P(c)。

推论（因式定理）：若 P(c) = 0，则 (x−c) 是 P(x) 的因式。

有理根定理 (Rational Root Theorem)

对整系数多项式 anxⁿ + … + a₀，所有可能的有理根形式为：

± (常数项 a₀ 的因子) / (首项系数 an 的因子)

5.6 解题路线图

  1. 猜第一个有理根（有理根定理，首选 ±1, ±2 试）。
  2. 用综合除法或长除法降次。
  3. 对商式反复猜根、降次，直到剩二次或一次。
  4. 二次式用十字相乘或求根公式收尾。

六、回文多项式与韦达定理反推

6.1 回文多项式（倒数方程）

系数左右对称的多项式。如 x⁴ + 2x³ + 3x² + 2x + 1（系数 1, 2, 3, 2, 1）。

性质：若 r 是根，则 1/r 也是根。奇数次回文多项式必有根 x = −1 或 x = 1。

6.2 y = x + 1/x 换元技巧

对于偶数次回文多项式，除以 xn/2，两两配对，设 y = x + 1/x。

  • y² = x² + 2 + 1/x² → x² + 1/x² = y² − 2
  • y³ = x³ + 3x + 3/x + 1/x³ → 可递推更高次

6.3 韦达定理（三次为例）

对 x³ + px² + qx + r，设根为 x₁, x₂, x₃：

x₁ + x₂ + x₃ = −p
x₁x₂ + x₂x₃ + x₃x₁ = q
x₁x₂x₃ = −r

6.4 等差数列根的因式分解

设三根为 a−d, a, a+d，则：

  • 3a = −p → a = −p/3
  • 3a² − d² = q → d² = 3a² − q = p²/3 − q
  • d = ±√d²，两种取法给出完全相同的三个根（只是排列顺序不同）

为什么 d 的正负不影响结果？因为多项式系数只含 d²（平方消去了符号），且 d 取正或负只是左右两侧的根互换了名字。

七、定积分计算：三个常见"卡脚点"与修复

7.1 问题一："对谁求导？"

积分变量是哑元，自变量在"边界"上。t 只在积分号内部跑一遍，跑完就消失。F 是 x 的函数，求导永远是对 x 求。

操作铁律：每次下笔前自问——被积函数里的变量是谁？（必是 t）积分上限/下限里有没有 x？上限是 x 还是 x 的函数？（决定要不要链式法则）

公式树：

d/dx ∫ax f(t)dt = f(x)

d/dx ∫ag(x) f(t)dt = f(g(x))·g'(x)

d/dx ∫h(x)g(x) f(t)dt = f(g(x))g'(x) − f(h(x))h'(x)

7.2 问题二：换元不换限

深层原因：换元不是"换字母"，而是构造了一个新函数（新映射）。原来的映射是 x ↦ f(g(x))g'(x)，新映射是 u ↦ f(u)。映射变了，定义域（上下限）当然要跟着变。

关键认知

哑变量替换（如 x→t）：映射不变，限不变。
变量代换（如 u=g(x)）：映射从 x↦f(g(x))g'(x) 变成 u↦f(u)，映射变了，限必须变。

忘记换限 = 用新地图却用旧坐标找路。

操作铁律：定积分换元必须三个动作同一行完成——换被积函数、换上下限、换积分变量。推荐全程新变量模式（模式 A）：换元后全部写成 u 的表达式，上下限换成 u 的限，直接算，算完就是数值，无需代回。

7.3 问题三：基本公式不熟

公式是肌肉记忆，不是大脑记忆。推荐每天 3 分钟"热身模式"：白纸左边默写求导公式，右边对应写积分公式，写不出就空着，3 分钟后翻书用红笔补全。连续一周，公式自然长在手上。

八、学习与笔记：如何让灵感转化率更高

8.1 灵感的高耗散本质

哈伯法合成氨转化率 10–15%，葡萄糖氧化只有 ~34% 变成 ATP，其余为热。

灵感也一样——大部分像废热一样，当时滚烫，过后消散。这不是表达能力的问题，而是灵感本身就是一种低熵、高耗散的东西。

8.2 提高"转化率"的笔记框架

原则一：区分"灵感"和"总结"

学的时候只允许自己留"痕迹"（一个词、一张图、一个箭头）。学完之后立刻花 5 分钟做一次"精炼"。

原则二：三行以内铁律

  1. 这是什么？（一句话）
  2. 它为什么对/为什么反直觉？（一句话）
  3. 它让我联想到什么？（一句话）

原则三：用"未来的自己能否看懂"验收

写完三行后问自己：如果我现在失忆了，只剩这句话，我能重新推导出那个灵感吗？如果不能，补一个"钩子"——一个具体例子、一个类比、一个操作步骤。

原则四：允许非语言的钩子

箭头图、游戏角色的技能名、数学符号序列、只有你懂的暗号——都可以。钩子的唯一要求是：它能重新触发你当时的思维状态。

一句话总结

接受灵感本来就会以热的形式散失（这是宇宙规律）。你要做的不是挽留全部热量，而是抓住那 34% 变成 ATP 的部分——精炼过的、可再点燃的、三行以内的钩子。
