一、二重积分在算什么

一句话：曲顶柱体的体积

  • 一重积分：曲线下的面积。把区间切成小段，面积 = 高度 × 底宽，累加。
  • 二重积分：曲面下的体积。把区域切成小方块，体积 = 高度 × 底面积，累加。

\iint_D f(x,y) \, d\sigma = \lim \sum_i \sum_j f(x_i^*, y_j^*) \Delta x_i \Delta y_j

若 f(x,y)=1，则二重积分 = 区域 D 的面积。若 f(x,y) \ge 0，则表示以 D 为底、曲面 z=f(x,y) 为顶的曲顶柱体体积。

二、X型与Y型区域

计算二重积分的方法是化为两次一重积分，有两种切法：

竖着切（X型：先 y 后 x）

用竖线穿过区域，先算一条的面积，再把所有条沿 x 累加：

\int_a^b \left[ \int_{y_1(x)}^{y_2(x)} f(x,y) \, dy \right] dx

横着切（Y型：先 x 后 y）

用横线穿过区域，先算一条的面积，再把所有条沿 y 累加：

\int_c^d \left[ \int_{x_1(y)}^{x_2(y)} f(x,y) \, dx \right] dy

关键：内层积分时，另一个变量当常数看。内层积完得到一个单变量函数，再对外层积分。

三、如何画出积分区域

以 ∫_0^1 ∫_x^1 sin(y²) dy dx 为例：

第一步：积分限 → 不等式组

0 ≤ x ≤ 1， x ≤ y ≤ 1

第二步：不等号改等号 → 三条边界线

x = 0（y轴）， y = x（45°斜线）， y = 1（上边）

第三步：三线交点 → 确定形状

  • x=0 与 y=x → (0,0)
  • x=0 与 y=1 → (0,1)
  • y=x 与 y=1 → (1,1)

三个交点连起来，是一个三角形。

第四步：交换次序（横着看）

y 从 0 到 1。固定 y，x 从 0 到 y（左边贴 y 轴，右边碰斜边）。

∫_0^1 ∫_0^y sin(y²) dx dy → ∫_0^1 y·sin(y²) dy = (1−cos 1)/2

四、交换次序的底层原理：同一个集合，两种描述

积分限的本质是用不等式描述一个集合：

D = \{(x,y) \mid 0 \le x \le 1,\ x \le y \le 1\}

交换次序就是换个方向重新描述同一个集合。从横着看：

D = \{(x,y) \mid 0 \le y \le 1,\ 0 \le x \le y\}

两个不等式组描述的是完全相同的点集。切法变了，集合没变，所以积分值不变——这就是富比尼定理的直觉版本。

五、提取出来的通用规律

  1. 积分限 = 不等式组：拿到二重积分立刻写成不等式，这是读懂区域的最快方式。
  2. 边界 = 不等式的等号：所有不等号改成等号就是边界线，画出线就能找到交点。
  3. 交换次序 = 重新解不等式：换个变量当外层，重新解出内层变量的范围。
  4. 切法变了，集合没变：无论竖切横切，切的始终是同一个集合。

六、第一次积分的被积函数是"切片"还是"整体"？

两者都是，只是视角不同。

  • 作为整体：f(x,y) 定义在整个区域 D 上，描述整个曲面。
  • 作为切片：固定 x 做内层积分时，f(x,y) 被"局部调用"为当前切片的高度函数。
一句话

f(x,y) 描述的是整体曲面，但在第一次积分时，它被"局部调用"为当前切片的高度函数。整体是它的定义，切片是它的用法。

七、z=1：当二重积分退化为面积

当 f(x,y)=1，曲面退化为高度恒为 1 的平面。第一次积分（先对 y）时：

\int_{y_1(x)}^{y_2(x)} 1 \, dy = y_2(x) - y_1(x) = \text{竖条长度}

外层积分把所有竖条长度沿 x 累加，得到区域面积：

\iint_D 1 \, dxdy = \text{区域 } D \text{ 的面积}

被积函数几何意义第一次积分算出
f(x,y) 任意曲顶柱体体积切片面积
f(x,y)=1区域 D 的面积切片长度（宽）
面积只是高度为 1 时的体积。理解 z=1 这个特殊情形，就能理解二重积分为什么可以同时用来算体积和面积。
