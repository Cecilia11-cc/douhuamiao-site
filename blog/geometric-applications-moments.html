一、几何应用 + 二重积分：8 题精练

题1：平面面积

y=x² 与 y=x+2 交点 x=−1,2。面积=∫₋₁²[(x+2)−x²]dx = 9/2。

题2：绕 x 轴旋转体

y=sin x (0≤x≤π) 绕 x 轴：V=π∫₀πsin²x dx = π²/2。

题3：壳层法——绕 y 轴，避免反解

y=ln x, x=e, x 轴围成，绕 y 轴。壳层法：V=2π∫₁e x·ln x dx = π(e²+1)/2。

题4：极坐标面积

心形线 r=1+cosθ。A=(1/2)∫₀2π(1+cosθ)² dθ = 3π/2。易错：漏 r 的平方。

题5：二重积分基本定限

∬D(x+y)dσ，D 由 y=x² 和 y=x 围成。结果 3/20。

题6：交换积分次序

∬D ey² dσ。ey² 对 y 不可积，必须交换次序：先 x 后 y，结果 (e−1)/2。

题7：极坐标二重积分

∬D√(1−x²−y²)dσ，D 为单位圆。极坐标变换，忘乘 r 会全错。结果 2π/3（半球体积）。

题8：二重积分求体积

平面 z=6−2x−3y 与坐标面围成四面体。V=∬D(6−2x−3y)dσ=6。可用几何公式 (1/6)×底面积×高 验证。

二、壳层法：剥洋葱式的体积计算

为什么需要壳层法？

绕 y 轴旋转时，圆盘法需要反解函数为 x=g(y)。当反解困难时，壳层法直接对 x 积分，避免反解。

几何直觉

在 x 处取一个宽 dx 的竖条，绕 y 轴旋转形成圆柱壳（空心圆筒）。把壳剪开、铺平，近似为一个长方体：

  • 长 = 周长 = 2πx
  • 宽 = 高度 = f(x)
  • 高（厚度）= dx

体积微元：dV = 2πx·f(x)·dx

V = ∫ab 2πx·f(x) dx

壳层法 vs 圆盘法速查

场景首选方法原因
绕 x 轴，y=f(x)圆盘法半径就是 f(x)
绕 y 轴，y=f(x)壳层法半径 x，高 f(x)，避免反解
绕 y 轴，x=g(y)圆盘法半径就是 g(y)
轴不是坐标轴视函数而定修正半径：距离=|f(x)−c|
函数难反解壳层法保持原变量
口诀：轴与自变量同向用圆盘，异向用壳层。难反解时壳层救急。轴平移时半径要修正。

三、极坐标面积：为什么是 r² 而不是 r？

你把每一小块近似为小三角形的直觉完全正确：

  • 底 = 弧长 = r·dθ（你已经理解了）
  • 高 = 半径 = r
  • 三角形面积 = (1/2)×底×高 = (1/2)×(r·dθ)×r = (1/2)r² dθ

你之前列式漏掉了平方，是因为只写了 r 而忘了高也是 r。公式：

A = (1/2) ∫αβ r² dθ

四、圆盘法 vs 壳层法：5 道决策训练

题1：y=x² (0≤x≤1) 绕 y 轴

两者均可。壳层法一步到位：V=∫₀¹ 2πx·x² dx = π/2。

题2：y=√x, x=4, x 轴 绕 x 轴

圆盘法最自然：V=π∫₀⁴ (√x)² dx = 8π。

题3：y=x², y=4 绕 y=4

轴不是坐标轴！圆盘法修正半径 R=4−x²。V=∫₋₂² π(4−x²)² dx = 512π/15。

题4：y=sin x (0≤x≤π) 绕 y 轴

壳层法经典案例。反解 x=arcsin y 需处理左右两支，极繁。壳层法一步：V=∫₀π 2πx·sin x dx = 2π²。

题5：y=eˣ, x=0..1, y=0 绕 y 轴

壳层法：V=∫₀¹ 2πx·eˣ dx = 2π。圆盘法需分两段积分。

选择决策树

  1. 旋转轴与积分变量同向→ 圆盘法（自然）
  2. 旋转轴与积分变量异向→ 壳层法（避免反解）
  3. 函数难反解→ 无条件壳层法
  4. 轴平移→ 先修正半径，再选方法

五、sin x 与 cos x：为什么体积不同？

你的发现

把 sin x 画成了 cos x，导致认为有部分在 x 轴下方。sin x 在 [0,π] 上全程 ≥0。

两倍左半边为什么不对？

  • 左半边体积：∫₀π/2 2πx·sin x dx = 2π
  • 右半边体积：∫π/2π 2πx·sin x dx = 2π²−2π
  • 总和：2π²
  • 2×左半边 = 4π ≠ 2π²

右半边离 y 轴更远，壳层半径更大，体积贡献远大于左半边。面积对称不代表体积对称。

cos x 能换成 sin x 算吗？

  • 有平方的项目（绕 x 轴体积、弧长）：平方消去正负，可以互换。
  • 有一次方+权重的项目（绕 y 轴壳层法）：不能换，必须按 |cos x| 分段。

六、对称性与"二极管思维"

你把自己的错误命名为"二极管思维"——只想到图形对称，忘了被积函数里的权重不对称。

关键区分

面积是"高度"的累积 ∫f(x)dx，权重均匀，对称可直接翻倍。
体积（壳层法）是"高度×半径"的累积 ∫x·f(x)dx，半径 x 是越来越大的权重，对称性被打破。
极坐标面积是 (1/2)∫r²dθ，r² 也是权重，但若 r(θ) 本身对称则 r² 也对称。

以后翻倍前先问：被积函数里的权重（x、r²、y）是否也对称？不对称就不能翻倍。

七、壳层法与垫圈法的加减逻辑

你问：右半边旋转产生的壳，是不是包含了左半边的空间？需不需要相减？

方法积分变量微元形状加减逻辑
壳层法x竖条→圆柱壳纯加（每个壳独立，互不重叠）
垫圈法y横条→垫圈外减内（同一高度处外圆面积减内圆面积）

壳层法像切洋葱：一层一层剥，每层有自己的体积，总和就是整个洋葱的体积。垫圈法像切甜甜圈：每个高度切一刀，得到一个圆环，面积是外圆减内圆。

八、矩：从零阶到四阶

你之前用"5个区间加权1,2,3,4,5"解释 sin x 和 cos x 的体积差异，这其实已经自己发现了矩的核心。

阶数名称公式描述什么概率对应
0阶总质量∫ρ dx总量（面积、质量）—
1阶静矩∫xρ dx分布的中心位置期望
2阶惯性矩∫x²ρ dx分布的"展宽"或抗转能力方差
3阶偏度∫x³ρ dx分布的不对称程度偏度
4阶峰度∫x⁴ρ dx分布的"尖峭"程度峰度

与旋转体体积的直接对应

壳层法体积 = 2π × 一阶矩（对 y 轴的静矩）。

绕 x 轴体积 = π × ∫[f(x)]² dx，涉及二阶矩。

极坐标面积 = (1/2) × 对极点的二阶矩。

直觉阶梯

零阶矩：只关心总量。
一阶矩：关心重心离轴多远。
二阶矩：更强烈地惩罚离轴远的点（平方权重）。
这个阶梯和你从"面积"到"体积"再到"极坐标面积"的学习路径完全一致。
