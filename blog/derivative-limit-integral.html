一、导数定义：一切推导的起点

函数 f(x) 在 x 处的导数，定义为：

f'(x) = limh→0 [f(x+h) − f(x)] / h

这是唯一的核心。所有求导公式都从这个极限推出来。

整个推导网络只需要四个"元初"极限：
  1. 导数定义的极限
  2. limh→0 (sin h)/h = 1（几何夹逼可证）
  3. limh→0 (e^h − 1)/h = 1（由 e 的定义推出）
  4. limh→0 (1+h)1/h = e（e 的定义）

二、四则运算的求导法则

和差法则：(f ± g)' = f' ± g'

直接由极限的线性性质推出——极限可分配进和差。

数乘法则：(c·f)' = c·f'

常数 c 可以从极限中提出。

乘积法则：(f·g)' = f'·g + f·g'

推导技巧——"加一项减一项"：

f(x+h)g(x+h) − f(x)g(x) = f(x+h)[g(x+h)−g(x)] + g(x)[f(x+h)−f(x)]

除以 h 取极限，f(x+h)→f(x)，即得乘积法则。

商的法则：(f/g)' = (f'g − fg')/g²

分子同样加一项减一项 f(x)g(x)，结构类似于乘积法则的变形。

三、链式法则（复合函数求导）

若 y = f(g(x))，则：

dy/dx = f'(g(x)) · g'(x)

直观推导：令 u = g(x)，则：

Δy/Δx = (Δy/Δu) · (Δu/Δx)

取极限 Δx→0，则 Δu→0（因 g 可导必连续），得：

dy/dx = (dy/du) · (du/dx)

严格推导需处理 Δu=0 的边界情况，但直觉如此。

四、基本初等函数的导数推导

常数函数：(c)' = 0

分子恒为 0，极限为 0。

幂函数：(x^n)' = nx^{n−1}

用二项式展开 (x+h)^n，减去 x^n 后除以 h，含 h 的项在 h→0 时全消失，仅剩 nx^{n−1}。此公式对实数 n 同样成立（可用对数求导证明）。

指数函数：(e^x)' = e^x

从定义出发：(e^{x+h}−e^x)/h = e^x·(e^h−1)/h。由 e 的定义，lim_{h→0}(e^h−1)/h = 1，故 (e^x)' = e^x。详见第五节。

对于一般指数：(a^x)' = a^x·ln a（将 a^x = e^{x·ln a} 用链式法则）。

对数函数：(ln x)' = 1/x（x > 0）

用导数定义 + e 的极限性质：令 t = h/x，极限化为 (1/x)·lim_{t→0} ln[(1+t)^{1/t}] = (1/x)·ln e = 1/x。

三角函数

sin x：用和角公式 sin(x+h)=sin x·cos h + cos x·sin h，配合 lim(sin h)/h = 1 和 lim(cos h−1)/h = 0，得 (sin x)' = cos x。详见第六节。

cos x：类似推导，得 (cos x)' = −sin x。

tan x：写为 sin x/cos x，用商法则，得 sec² x。

反三角函数

用反函数法则或隐函数求导：

arcsin x：令 y=arcsin x 即 x=sin y，对 x 求导得 1=cos y·dy/dx，cos y=√(1−x²)，故 (arcsin x)' = 1/√(1−x²)。

arctan x：令 y=arctan x 即 x=tan y，1=sec²y·dy/dx，sec²y=1+x²，故 (arctan x)' = 1/(1+x²)。

五、e^x 导数推导的完整逻辑链

e 的两种等价定义

定义一（复利极限）

e = limn→∞ (1 + 1/n)n = limh→0 (1 + h)1/h

定义二（导数定义——你要推导的目标）

e 是唯一满足下式的正数：limh→0 (e^h−1)/h = 1

这个定义说：指数函数 e^x 在 x=0 处的导数是 1。

从定义一到定义二的推导

设 t = e^h − 1，则 e^h = 1+t，h→0 时 t→0。

两边取自然对数：h = ln(1+t)。代入极限：

limh→0 (e^h−1)/h = limt→0 t/ln(1+t) = limt→0 1/ln[(1+t)1/t]

根据定义一：limt→0 (1+t)1/t = e，所以极限 = 1/ln e = 1。

完整逻辑链

定义：e = lim(1+1/n)^n  →  换元得 lim(e^h−1)/h = 1  →  (e^x)' = e^x
直觉视角

当 h 极小时，e^h ≈ 1+h（e 是"单位增长率下连续复利的极限"），所以 (e^h−1)/h ≈ (1+h−1)/h = 1。取极限后近似变精确。e 的本质就是：它是使得指数函数在 0 处的变化率恰好为 1 的唯一底数。e^x 的导数还是它自己——这是 e 被选中的"使命"。

六、sin x 的推导：符号修正与极限拆分

正确的和角公式

sin(x+h) = sin x·cos h + cos x·sin h

注意第二项是 + cos x·sin h，不是 −sin h·cos x。

推导

sin(x+h) − sin x = sin x(cos h − 1) + cos x·sin h

除以 h 取极限：

(sin x)' = sin x·limh→0(cos h−1)/h + cos x·limh→0(sin h)/h

两个关键极限：

  • 极限一：limh→0 (sin h)/h = 1（几何夹逼可证）
  • 极限二：limh→0 (cos h−1)/h = 0（可化为 −(sin h)/h · sin h/(cos h+1) → −1·0/2 = 0）

代入得：(sin x)' = sin x·0 + cos x·1 = cos x。

七、极限拆分的合法性：什么时候能"先算一部分"？

核心规则

极限加法法则：若 lim A(h) 和 lim B(h) 都存在且为有限数，则 lim[A+B] = lim A + lim B。

你担心的情况分析

对第一项 sin x(cos h−1)/h 单独取极限得 0——这完全合法，因为该极限存在且为有限数（0）。对第二项 cos x·sin h/h 单独取极限得 cos x——同样合法。两项极限都存在且有限，和自然就是 0+cos x = cos x。

什么时候这样做会出错？

如果某项的极限不存在（或为无穷大），就不能先算完它再处理另一项。

反例：limx→0(1/x − 1/x)。如果"先对第一项取极限"得 ∞，然后继续算，就会出错。实际上两项先通分再取极限，结果是 0。这就是 ∞−∞ 不能拆的原因。

进阶意识

你做的不是"随便把一个部分先变成 0"，而是：利用极限加法法则 → 拆成两个独立极限 → 分别算出 0 和 cos x → 相加。每一步条件都满足，完全合法。你能主动审视"拆开求极限的合法性"，说明你已经开始建立数学推导的"规则意识"了。

九、10 道极限找茬题

每题给出了一种计算过程，先判断是否正确；如果错误，指出错在哪一步和为什么。

题1

求 limx→0 (1/x − 1/sin x)

错误解法：拆成 lim 1/x − lim 1/sin x = ∞−∞ = 0

错误 拆分时两项都不存在（∞），不能直接相减。应先通分，再用洛必达或泰勒。结果为 0。

题2

求 limx→0 (sin x − x)/x³

错误解法：sinx~x，所以 sinx−x~0，极限为 0。

错误 在加减法中直接对单项使用等价无穷小替换。正确用泰勒 sinx = x−x³/6+o(x³)，结果为 −1/6。也可用三次洛必达。

题3

求 limx→0 x·sin(1/x)

可疑解法：拆成 lim x · lim sin(1/x) = 0 × 不存在 = 0。结果对但过程对吗？

过程错误 第二步拆开不合法，因为 lim sin(1/x) 不存在。正确做法：用无穷小乘有界函数，|x·sin(1/x)| ≤ |x| → 0，夹逼定理得 0。

题4

求 limx→∞ (1+1/x)2x

错误解法：先取底数极限得 1，所以 12x=1。

错误 这是 1^∞ 型未定式，不能先取底数极限。正确：[(1+1/x)^x]² → e²。

题5

求 limx→∞ x/(x+sin x)

错误解法：洛必达得 1/(1+cos x)，该极限振荡不存在，所以原极限不存在。

错误 洛必达要求求导后极限存在或为无穷才能推出原极限。求导后振荡不满足条件。正确做法：分子分母同除 x，得 1/(1+sin x/x)，sin x/x→0，极限为 1。

题6

求 limx→0 (1+x)1/x

错误解法：1+x→1，所以 1^∞=1。

错误 典型 1^∞ 型。正确利用重要极限：limx→0(1+x)1/x = e。

题7

求 limx→0 (tan x − sin x)/x³

错误解法：tanx~x, sinx~x，所以 tanx−sinx~x−x=0，极限为 0。

错误 同题2，加减法中误用等价无穷小。正确：tanx−sinx = tanx(1−cosx) ~ x·x²/2 = x³/2，结果 1/2。

题8

求 limx→0 (e^x−1−x)/x²

混合解法：洛必达一次后得 (e^x−1)/2x ~ x/2x = 1/2。这种等价替换是否允许？

允许 因为洛必达后 (e^x−1)/2x 是一个整体分式，等价替换发生在乘除因子中。本题结果正确，用来训练判断力——不是所有"看起来像"的替换都是错的。

题9

求 limx→0 x/|x|

错误解法：约掉绝对值，极限为 1。

错误 绝对值不能随意约掉。x→0⁺ 时极限为 1，x→0⁻ 时极限为 −1，左右极限不等，极限不存在。

题10

求 limx→0 ∫0x sin t dt / x²

常见错误：∫0x sin t dt = −cos x（忘了代入下限 0）。

错误 定积分不是不定积分。正确：∫0x sin t dt = [−cos t]0x = 1−cos x。然后原式 = lim(1−cos x)/x² = 1/2（洛必达或等价替换）。

拿到极限题的第一反应不是动手算，而是先判断：是什么型？拆分条件满足吗？等价能换吗？左右极限要分吗？

十、基本积分公式练习：8 道题

题目

  1. ∫ (3x² − 2/x + 1) dx
  2. ∫ (e^x + 2^x) dx
  3. ∫ (sin x + cos x) dx
  4. ∫ sec² x dx
  5. ∫ 1/(1+x²) dx
  6. ∫ 1/√(1−x²) dx
  7. ∫ tan x dx
  8. ∫ 1/(x²−4) dx

逐题详解

题1：∫ (3x² − 2/x + 1) dx

  1. 分项：∫ 3x²dx − ∫ 2/x dx + ∫ 1dx
  2. 第一项：3·x³/3 = x³
  3. 第二项：2·ln|x|（注意绝对值——1/x 的积分是 ln|x|，因为负半轴也有定义）
  4. 第三项：x

结果：x³ − 2ln|x| + x + C

检查：对结果求导，得 3x² − 2/x + 1，匹配。

题2：∫ (e^x + 2^x) dx

e^x + 2^x/ln 2 + C

口诀：e^x 积分不变；a^x 积分除以 ln a。易错：忘了除 ln 2。

题3：∫ (sin x + cos x) dx

−cos x + sin x + C

题4：∫ sec² x dx

tan x + C（因为 (tan x)' = sec² x）

题5：∫ 1/(1+x²) dx

arctan x + C（因为 (arctan x)' = 1/(1+x²)）

题6：∫ 1/√(1−x²) dx

arcsin x + C（因为 (arcsin x)' = 1/√(1−x²)）

题7：∫ tan x dx —— 最需要拆解的一题

  1. 写成 sin x/cos x
  2. 设 u = cos x，du = −sin x dx，即 sin x dx = −du
  3. 积分变为 −∫ 1/u du = −ln|u| + C
  4. 代回：−ln|cos x| + C = ln|sec x| + C

易错：忘了负号会写成 ln|cos x|（符号反）；忘了绝对值。

题8：∫ 1/(x²−4) dx

用部分分式：1/(x²−4) = 1/4(x−2) − 1/4(x+2)。积分得：

(1/4)ln|x−2| − (1/4)ln|x+2| + C = (1/4)ln|(x−2)/(x+2)| + C

通式：∫ 1/(x²−a²) dx = (1/2a) ln|(x−a)/(x+a)| + C，这里 a=2。
