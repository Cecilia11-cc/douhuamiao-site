一、第一轮：进阶题组

核心工具不变——公式、代数变形、换元、分拆。心法：先观察结构 10 秒，判断类型，再动笔。

  1. ∫ x√(x²+1) dx
  2. ∫ eˣ/(1+e²ˣ) dx
  3. ∫ sin²x dx
  4. ∫ 1/(x²+4x+13) dx
  5. ∫ x ln x dx
  6. ∫ 1/[√x(1+x)] dx
  7. ∫ x²/√(1−x²) dx
  8. ∫ eˣ sin x dx

二、第一轮逐题详解

题1：凑微分

令 u=x²+1，du=2xdx。∫ x√(x²+1)dx = (1/2)∫√u du = (1/3)(x²+1)3/2+C。

题2：换元 + arctan

令 u=eˣ，du=eˣdx。∫ eˣ/(1+e²ˣ)dx = ∫ 1/(1+u²)du = arctan(eˣ)+C。注意结果写成 arctan(eˣ)，加括号。

题3：降幂公式

sin²x=(1−cos2x)/2。∫ sin²x dx = x/2 − sin2x/4 + C。易错：∫ cos2x dx 需补系数 1/2。

题4：配方法

分母=(x+2)²+9，套公式 ∫ 1/(u²+a²)du = (1/a)arctan(u/a)+C，结果 (1/3)arctan((x+2)/3)+C。

题5：分部积分

u=lnx, dv=xdx。结果 x²/2·lnx − x²/4 + C。口诀：对数反三角优先选作 u。

题6：换元消根

令 u=√x，x=u², dx=2udu。∫ 2/(1+u²)du = 2arctan(√x)+C。

题7：三角换元

令 x=sinθ，化成 ∫ sin²θ dθ，与题3同。结果 (1/2)arcsin x − (x/2)√(1−x²)+C。

题8：分部积分循环

两次分部积分，产生循环方程。结果 (eˣ/2)(sin x−cos x)+C。

三、第一轮错题复盘

题2：漏括号

写成了 arctan eˣ 而不是 arctan(eˣ)。虽无歧义，但养成习惯：凡是函数的自变量是复合表达式，加括号。代入上下限时括号能防连环错。

题3 与题7：发现它们是同一题

题3 ∫ sin²x dx，题7三角换元后变成 ∫ sin²θ dθ。这是结构眼光的突破。卡住原因：对题3没用降幂，直接硬上分部积分——虽能做出来但多绕很多弯。

教训：遇到 sin²x, cos²x，先降幂再积分，这是肌肉记忆。遇到 √(1−x²)，先三角换元，也是肌肉记忆。

题4：换元后忘记"回家"

令 x+2=3tan u，积分得 u/3+C。然后写了 arctan(tan u)/3 并错误简化成 (x+2)/3。断档处：u 和 x+2 不是一回事——应从 tan u=(x+2)/3 回代，得 u=arctan((x+2)/3)。

心法：每次换元，在草稿上写清楚映射——原变量→新变量，新变量→原变量。算完后对着映射换回去。

题5：分部积分第一步跳步

方向对但漏了系数。分部积分三步不可跳：明确写下 u,dv → 算出 du,v → 代入 uv−∫v du。

四、第二轮：强化题组

特点：单一技巧不够用，需要组合技。

  1. ∫ ln x/x² dx
  2. ∫ x arctan x dx
  3. ∫ 1/(x³+x) dx
  4. ∫ √(1−x²) dx
  5. ∫ 1/sin x dx = ∫ csc x dx
  6. ∫ 1/(√x+∛x) dx
  7. ∫₀π x sin x dx（定积分）
  8. ∫ e²ˣ/(eˣ+1) dx

五、第二轮逐题详解

题1：分部积分

u=lnx, dv=x⁻²dx。v=−1/x。结果 −(ln x+1)/x + C。

题2：分部积分 + 有理化

u=arctanx, dv=xdx。剩余 ∫ x²/(1+x²)dx = ∫(1−1/(1+x²))dx = x−arctanx。结果 (x²/2)arctanx − x/2 + (1/2)arctanx + C = (x²+1)/2·arctanx − x/2 + C。

题3：有理函数拆分

分母=x(x²+1)，拆分得 1/x − x/(x²+1)。积分：ln|x| − (1/2)ln(x²+1) + C = ln|x/√(x²+1)| + C。

题4：三角换元 + 降幂

令 x=sinθ，化成 ∫ cos²θ dθ。降幂 cos²θ=(1+cos2θ)/2。结果 (1/2)arcsin x + (x/2)√(1−x²) + C。

题5：经典 csc x

乘 (csc x+cot x)/(csc x+cot x)，分子恰好是分母导数的相反数。令 u=csc x+cot x，积分=−ln|u|+C=−ln|csc x+cot x|+C。等价形式：ln|tan(x/2)|+C。

题6：找最小公倍数代换

令 x=t⁶（6=2×3），dx=6t⁵dt。∫ 6t⁵/(t³+t²)dt = 6∫ t³/(t+1)dt。多项式除法：t³=(t+1)(t²−t+1)−1。结果 2√x−3∛x+6x1/6−6ln(x1/6+1)+C。

题7：定积分分部积分

u=x, dv=sinx dx。∫₀π x sinx dx = [−x cosx]₀π + ∫₀π cosx dx = π + 0 = π。

题8：指数分解

巧法：e²ˣ=eˣ(eˣ+1)−eˣ。∫ eˣ dx − ∫ eˣ/(eˣ+1)dx = eˣ − ln(eˣ+1)+C。

六、第二轮错题复盘

题2：最后一步抄错

把 x² 抄成了 x。纯属书写问题，但说明验算环节缺失。做完后求导回代可以捕获这类低级错误。

题3：困在 1/(1+x²) 和 1/x 之间

一个积分含两种复杂元素——1/x 和 1/(1+x²)——做出来的概率很低，因为它们不是同类。你复盘时自己总结出了这一点：非同类的复杂元素组合，用拆分来处理，这正是有理函数拆分的触发条件。

题4：降幂公式记反

cos²x=(1+cos2x)/2，你记成了 (1−cos2x)/2。这是纯粹的公式记忆问题。快速验证：代 x=0，cos²0=1，(1+cos0)/2=1 正确，(1−cos0)/2=0 错误。

题5：不知道"乘共轭"的技巧

∫ csc x dx 的标准做法是乘 (csc x+cot x)/(csc x+cot x)。这个技巧不是凭空想出来的——它来自观察到 (csc x+cot x) 的导数恰好是 −csc x(csc x+cot x) 的某种形式。如果不知道这个技巧，另一条路是万能半角代换——令 t=tan(x/2)，将 sin x 和 cos x 都表为 t 的有理函数，也能做。

思考路径：遇到三角有理函数积分，先试凑微分；不行就用万能半角代换 t=tan(x/2)，它能将任何 sin x, cos x 的有理式化为 t 的有理函数。

题6：理想思路

先验证两个根号能否合并或消掉→发现不能→分开研究→观察到 √x 和 ∛x 的最小公倍数是 6→令 x=t⁶。这是标准的"根式换元"触发逻辑。

题7：u 的选择策略

你把 x 当 u，sin x 当 dv——这其实是正确的选择！口诀是"对数反三角优先选作 u"，但 x 不是对数也不是反三角，所以优先权不在它身上。这里的策略是：让求导后变简单的当 u，积分后不变复杂的当 dv。

对于 ∫ x sin x dx：x 求导变 1（变简单了），sin x 积分变 −cos x（不会变得更复杂），所以选 u=x 完全正确。你之前对自己策略的怀疑其实不必要——你选对了。

七、第六题：换两次元完全可行

原题6：∫ x/(√(1+x²)−√(1−x²)) dx。你的思路：有理化→拆成两项→第一项令 x=tanθ→第二项令 x=sinα。

完整过程

有理化后：

I = (1/2)∫ √(1+x²)/x dx + (1/2)∫ √(1−x²)/x dx = (1/2)(I₁+I₂)

I₁：令 x=tanθ

化成 ∫ sec³θ/tanθ dθ = ∫ (sinθ/cos²θ + cscθ)dθ = secθ − ln|cscθ+cotθ|+C。

回代得：√(1+x²) − ln|(√(1+x²)+1)/x| + C。

I₂：令 x=sinα

化成 ∫ cos²α/sinα dα = ∫ (cscα−sinα)dα = −ln|cscα+cotα| + cosα + C。

回代得：√(1−x²) − ln|(1+√(1−x²))/x| + C。

合并

I = (1/2)[√(1+x²)+√(1−x²) − 对数项] + C

总结

分拆 + 双换元完全正确，步骤长但可靠性高。考试中如果找不到巧解，这就是最稳的策略。代价是回代和合并对数时容易出错，但只要仔细，一定能做对。

八、cot·csc 与 cos·cot：极易混淆的一对

你混淆的地方

在 I₂ 中出现了 ∫ cos²α/sinα dα。你把它看成了 ∫ cotα·cscα dα，但：

  • cotα·cscα = (cosα/sinα)·(1/sinα) = cosα/sin²α
  • cos²α/sinα = cosα·(cosα/sinα) = cosα·cotα

分母 sin 的次数差了一次，不相等。

正确的路径

正因为分子是 cos²α，才用恒等式 cos²α=1−sin²α 拆开：

∫ cos²α/sinα dα = ∫ (1−sin²α)/sinα dα = ∫ cscα dα − ∫ sinα dα

速查：三角积分中的两对导数

  • (csc x)' = −csc x·cot x → ∫ cot x·csc x dx = −csc x + C
  • (sec x)' = sec x·tan x → ∫ sec x·tan x dx = sec x + C
防错心法：以后看到三角分式，先化成纯 sin, cos，再判断能否凑微分或拆开，不要直接凭"感觉"匹配公式。

九、分部积分：u 的选择策略

你提出的反思

"是不是应该把变化速率慢的当成 u？比如有 ln x 就把 ln x 当作 u，有 sin x 就把 sin x 当成 u。"

更精确的规则

分部积分选 u 的标准口诀——LIATE 原则（按优先级递减）：

优先级类型原因例
1（最高）Log 对数求导后变 1/x，大幅简化ln x
2Inverse trig 反三角求导后变代数函数arctan x, arcsin x
3Algebraic 代数（幂函数）求导降次，逐渐简化x, x²
4Trig 三角求导/积分循环，不简化sin x, cos x
5（最低）Exponential 指数积分不变复杂，求导也不简化eˣ

应用到具体题

  • ∫ x ln x dx：L 优先 → u=lnx, dv=xdx。正确。
  • ∫ x arctan x dx：I 优先 → u=arctanx, dv=xdx。正确。
  • ∫ x sin x dx：A 的优先级高于 T → u=x, dv=sinx dx。正确。
  • ∫ eˣ sin x dx：T 和 E 同级，都可以，选哪个都会进入循环。选 u=sinx 或 u=eˣ 都对。
你的直觉是对的

"变化速率慢的当 u"本质上是 LIATE 的朴素表达——对数求导变 1/x（速率骤降），反三角求导变代数（速率降），幂函数求导降次（速率慢降）。你的直觉和正式规则完全一致。

近几次暴露出的积分薄弱点汇总

  1. 降幂公式记反——用代入验证法（代 x=0）快速自查
  2. 换元回代断档——强制写"回代映射"草稿
  3. 分部积分跳步——强制三行：u/dv → du/v → uv−∫v du
  4. 三角恒等式混淆——先化纯 sin/cos 再判断
  5. 书写规范——函数复合加括号，系数不抄错
  6. 验算缺失——每道不定积分做完，求导回去检查
