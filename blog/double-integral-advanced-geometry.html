一、累次积分：逐步增大难度

底部区域固定在 [0,1]×[0,1]，只换顶面 z=f(x,y)：

#f(x,y)考点结果
1z=1基准：体积=底面积1
2z=x只依赖一个变量，直接乘区间长度1/2
3z=x+y可分离变量的线性组合1
4z=xy可分离变量1/4
5z=x²+y²多项式逐项积分2/3
6z=e^{x+y}指数可分离：e^{x+y}=eˣeʸ(e-1)²
9z=max(x,y)分段函数，须拆区域2/3
10z=1/(1+x²)结果含π：arctan出现π/4

题9详解：max(x,y) 在正方形上的积分

正方形被 y=x 分成两块。下三角(y≤x)：max=x；上三角(y≥x)：max=y。两块对称，算一块乘2。

∫_0^1 ∫_0^x x dy dx = ∫_0^1 x² dx = 1/3，两倍 = 2/3

二、区域边界辨析：x=y² 还是 x=√y？

这两个不是同一个函数，它们互为反函数，关于 y=x 对称。在 [0,1] 内，√y > y²。

题例：0≤y≤1, y²≤x≤y

边界：x=y²（抛物线）和 x=y（直线）。交点 (0,0),(1,1)。

类型外层内层下/左界上/右界
Y型（原题）y:0→1xx=y²x=y
X型（换序）x:0→1yy=xy=√x
规则：积分限里的变量必须是当前积分层的变量。外层是y→内层限用y表示；外层是x→内层限用x表示。边界方程的形式跟着积分次序走，不是固定的。

题4反过来：0≤x≤1, x²≤y≤√x

类型外层内层下/左界上/右界
X型（原题）x:0→1yy=x²y=√x
Y型（换序）y:0→1xx=y²x=√y

两种区域形式完全对称，只是变量名互换。

三、对称简化技巧

当区域关于 y=x 对称（边界互为反函数）时：

被积函数特征可用简化例
只含一个变量内层直接乘区间长度∫∫ x dσ → ∫ x(√x−x²)dx
f(x,y)=f(y,x)（对称）只算一半乘2x²+y²
f(x,y)=−f(y,x)（反对称）直接=0，不用算x−y → 0
对y积分困难把y全换成x∫∫ y³ dσ = ∫∫ x³ dσ

y=x²与y=x绕x=1。壳层法，半径修正为1−x。V=π/6。

五、锥体 vs 抛物面：一次方还是二次方

令 r=√(x²+y²)，看 z 与 r 的关系：

方程r-z关系类型剖面
z=1−r一次：r=1−z圆锥直线
z=1−r²二次：r=√(1−z)旋转抛物面抛物线

为什么一次是锥？

半径随高度均匀变化：z=1时r=0（尖点），z=0时r=1（底面）。截面圆从0均匀长到1，外轮廓是直线。

口诀：一次方是锥，二次方是碗。r=z是正锥（向上开口），r=1−z是倒锥（向下开口）。和抛物线的开口规律一致——系数符号决定方向。

为什么母线只需要半支？

旋转体母线只需 r≥0 那一侧。y=√z 只是 z=y² 的上半支，但绕z轴旋转后自动补全对称。旋转这个动作决定截面永远是圆。

六、母线 + 旋转：生成一切旋转体

核心公式：母线在yOz平面 y=f(z), z∈[a,b]，绕z轴旋转→ r=f(z)，即 x²+y²=[f(z)]²。

母线形状旋转体方程
y=1竖直线圆柱r=1
y=z斜直线(45°)圆锥(正)r=z
y=1−z斜直线(负斜率)圆锥(倒)z=1−r
y=√z半支抛物线旋转抛物面(碗)z=r²
y=z²抛物线旋转抛物面(细腰)r=z²
y=√(1−z²)四分之一圆弧半球r²+z²=1
y=e^z指数曲线喇叭形r=e^z
y=sin z波浪线花瓶形r=sin z

创新题：你来生成

  1. 母线 y=sin z (z∈[0,π]) 绕z轴 → 花瓶形，r=sin z
  2. 母线 y=2 (z∈[0,1]) 绕z轴 → 圆柱面，r=2
  3. 母线 y=1/z (z∈[1,3]) 绕z轴 → 旋转双曲面，r=1/z
  4. 拼接母线：y=1+z (z∈[0,1]) + y=2 (z∈[1,2]) → 高脚杯

术语澄清

术语定义旋转体中柱面中
母线运动生成曲面的线在平面内的曲线直线
准线母线运动所沿的固定曲线（不直接出现）椭圆/抛物线/双曲线
旋转轴母线绕其旋转的直线坐标轴（不存在）

七、准线 + 平移：柱面的统一原理

方程缺哪个变量，母线就平行于哪个坐标轴，准线就是另外两个变量构成的平面曲线。

准线母线方向生成曲面方程特征
圆 x²+y²=1平行于z轴圆柱面缺z
椭圆 x²/a²+y²/b²=1平行于z轴椭圆柱面缺z
抛物线 y=x²平行于z轴抛物柱面缺z
双曲线 x²/a²−y²/b²=1平行于z轴双曲柱面缺z
区分旋转体与柱面

旋转体：母线绕轴转，截面是圆，方程有 x²+y²。
柱面体：母线平移，截面是准线形状，方程缺一个变量。

扩展到二次曲面

如果旋转但母线换成椭圆/双曲线：

母线旋转轴生成曲面
椭圆 y²/b²+z²/c²=1z轴旋转椭球面
双曲线 y²/b²−z²/c²=1z轴单叶旋转双曲面
双曲线 z²/c²−y²/b²=1z轴双叶旋转双曲面
